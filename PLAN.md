# План развития personal page

> Сайт: **https://artfaal.ru**
> Репозиторий: `artfaal/artfaal.ru`
> Хостинг: GitHub Pages + Cloudflare DNS

---

## Принятые решения

- **Архитектура:** статический HTML + vanilla JS + CSS (без React, без build-step)
- **Страницы:** `index.html` (личное) + `cv/index.html` (работа/CV, URL: `/cv/`)
- **Контент:** единый `content.js` с комментариями — источник правды для обеих страниц
- **PDF:** отдельный скрипт `generate-cv.js` (Node.js + Chrome headless), результат в `assets/`
- **Двуязычность:** архитектура готова (CONTENT.ru / CONTENT.en), переводим когда будет готов русский текст
- **Тесты:** `node test.js` (67 проверок), запускаются автоматически при коммите
- **Pre-commit hook:** тесты + авто-перегенерация PDF при изменении content.js
